Xcode 15 Vim Commands Cheatsheet

Normal Mode Commands

  • Motion: h, j, k, l - Move cursor
  • Scrolling: Ctrl+d/u - Half page down/up
  • Editing: x, dd, yy - Delete char/line, yank line
  • Inserting: i, a, o, O - Insert mode entries
  • Search: /, ? - Search forward/backward

New & Improved in Xcode 15

  • gd - Jump to definition
  • Angle brackets support
  • % in Visual mode - Jump to counterpart

Visual Mode Commands

  • v, V, Ctrl+v - Visual mode variations
  • Text Objects: aw, iw - A/inner word

Insert Mode Commands

  • Esc - Return to Normal mode
  • Ctrl+r {reg} - Paste from register
