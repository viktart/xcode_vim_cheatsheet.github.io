Vim Commands in Xcode 15 Beta

Normal Mode Commands

Motion Commands

  • `h`, `l`, `k`, `j` and arrow keys for basic cursor movement.
  • `0`, `^`, `$` for start, first non-blank character, and end of line.
  • `F{character}`, `f{character}`, `T{character}`, `t{character}`, `;`, `,` for character search within a line.
  • `b`, `w`, `e`, `ge` for word navigation.
  • `{`, `}` for paragraph navigation.
  • `%` for matching bracket jump.
  • `gd` to jump to the definition of a symbol.

Scrolling Commands

  • `^y`, `^e` to scroll viewport up/down by one line.
  • `^u`, `^d`, `⌃b`, `⌃f` for half/full-page scrolls.
  • `z↩`, `zt`, `zz`, `z.`, `z-`, `zb` to reposition viewport with cursor line.

Insert and Replace Commands

  • `i`, `I`, `a`, `A`, `o`, `O` to enter insert mode.
  • `r` for character replacement.

Deleting Text Commands

  • `x`, `dd`, `D`, `d{motion}` for deleting characters or lines.
  • `J`, `gJ` for joining lines.
  • `c{motion}`, `cc`, `C`, `s`, `S` for change operations.

Copying and Moving Text Commands

  • `yy`, `y{motion}`, `Y` to yank (copy) text.
  • `p`, `P` for pasting.

Undo and Redo Commands

  • `u`, `⌃r` to undo and redo changes.

Visual Commands

  • `v`, `V`, `^v` to enter visual modes.
  • `%` now works in visual mode for jumping to counterpart.
  • `C-u` and `C-d` with count prefix in visual mode for quick scroll and selection adjustment.

Search Commands

  • `?`, `/` for searching.
  • `n`, `N` to repeat search.

Angle Bracket Block Support

  • Support for angle bracket block commands (`<`, `>`) for copying, modifying, or deleting contents.

Change Case Commands

  • `gu{motion}` and `gU{motion}` now change the case of characters rather than whole words.
